# 1.导论：浅谈SNN
 人脑：算力高、功耗低、学习快
## 1.1 神经元结构
树突-胞体-轴突-髓鞘
## 1.2 神经元的生理机能
1. 静息电位-达到阈值-去极化-复极化-超极化-恢复极化
2. 动作电位是离散事件
## 1.3 特性
1. 全有或全无
2. 不可逆性
3. 不应期存在
4. 主动传播
## 1.4 神经元数据模型——Hodgkin-Huxley模型
![[Pasted image 20260412231919.png]]
每模拟$1ms$需要进行1200次浮点运算
![[Pasted image 20260412232117.png]]
共同目标是模拟生物神经元的放点行为，即：
1. 建模神经动力学系统：通过微分方程表示电压电流和恢复变量的关系
2. 膜电位积累与阈值触发：再现膜电位在时间上的积分，当超过阈值时触发脉冲

## 1.5 脉冲神经网络（Spiking Neural Network）
是一类以脉冲神经元为基础，通过膜电位积累与阈值触发机制实现时序信息编码与传播的神经网络架构

| 模型名称                              | 简要介绍                                | 主要特点             |
| --------------------------------- | ----------------------------------- | ---------------- |
| ==LIF(Leaky Integrate-and-Fire)== | 最简单的脉冲神经元模型，模拟神经元电压积累和泄露过程，当超过阈值时放电 | 计算开销low，生物真实性low |
| Adaptive Exponential LIF          | LIF+指数上升和适应性电流，更贴近生物行为              | 平衡计算复杂度&生物真实性    |
| 二阶LIF模型                           | 增加对电压变化速率的建模                        | 表现更复杂的放电模式       |
| ==Izhikevich模型==                  | 用四个参数即可模拟多种生物神经元放电模式，效率高            | 生物真实性高、计算效率也高    |
